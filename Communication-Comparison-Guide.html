🔄 Communication Comparison Guide

STAR vs Pyramid vs Narrative Arc vs Executive Summary. Which framework for which situation? Quick reference + interactive picker.

⚡ One-Line Each

⭐
STAR Method
Situation → Task → Action → Result. Structure interview answers as 90-second stories.
🔺
Pyramid Principle
Answer first, then grouped reasons, then evidence. Top-down for efficiency.
📖
Narrative Arc
Status Quo → Disruption → Struggle → Resolution → New Reality. Build buy-in through tension.
📋
Executive Summary
One-page decision doc: Recommendation, Context, Data, Options, Risks, Ask.

🔀 Quick Decision Flow

What are you communicating?
Answering an interview question?
→ STAR
Writing a doc for leadership?
→ Exec Summary
Presenting & need buy-in?
→ Narrative Arc
Updating, emailing, or Slacking?
→ Pyramid

🔗 Power Combinations

Presentation + Follow-up: Present with Narrative Arc in the meeting, then send the Exec Summary as a follow-up doc. Spoken story + written one-pager = complete coverage.
Interview prep: Prepare 6-8 STAR stories. For case interviews, structure your analysis with Pyramid (conclusion first, grouped reasons).
Inside an Exec Summary: Use Pyramid structure for the Context and Key Data sections. The Exec Summary IS a Pyramid in document form with added Options and Risks.
Portfolio case studies: Write each project as a STAR story with an Exec Summary header. Recruiters scan the summary; interviewers hear the story.
